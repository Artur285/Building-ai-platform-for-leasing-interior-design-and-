# AI Platform for Interior Design Leasing

An intelligent platform that leverages AI to revolutionize interior design leasing and consultation services.

## Overview

This platform combines artificial intelligence with interior design expertise to provide:
- AI-powered design recommendations
- Virtual space planning and visualization
- Automated leasing management
- Style matching and preference learning
- Cost optimization for furniture and decor leasing

## Features

- **AI Design Assistant**: Intelligent recommendations based on space, style preferences, and budget
- **Virtual Room Designer**: 3D visualization and planning tools
- **Leasing Management**: Streamlined furniture and decor rental processes
- **Style Profiling**: Machine learning-based preference analysis
- **Inventory Management**: Smart catalog and availability tracking

## Getting Started

### Prerequisites

- Python 3.8+
- Node.js 16+
- PostgreSQL 12+

### Installation

```bash
# Clone the repository
git clone https://github.com/Artur285/Building-ai-platform-for-leasing-interior-design-and-.git
cd Building-ai-platform-for-leasing-interior-design-and-

# Install dependencies
pip install -r requirements.txt
npm install

# Set up environment variables
cp .env.example .env

# Run database migrations
python manage.py migrate

# Start the development server
npm run dev
```
